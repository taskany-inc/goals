![](/VersaCriteria.png)

### Настройки приватности проектов

В настройках проекта доступна секция приватности проекта. В этой секции указывается список участников проекта.

- Если список участников пуст, то проект является публичным;
- Если в списке есть хотя бы один учасник, то проект становится доступным только указанным пользователям

![](/PrivacyProjectSettings.jpeg)
